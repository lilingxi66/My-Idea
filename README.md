# .lst 文件生成器

这是一个简单的图形化工具，用于扫描指定的目录并生成包含文件名列表的 `.lst` 文件。

## 功能

- **纯图形界面 (GUI)**：提供直观的操作界面。
- **目标目录选择**：选择要扫描的文件夹。
- **保存目录选择**：自定义 `.lst` 文件的存放位置。
- **自定义输出文件名**：设置生成的列表文件名。
- **递归扫描**：支持深入子目录进行完整扫描。
- **自动排序**：生成的文件列表按字母顺序排列。
- **项目 Logo**：拥有专属的项目图标。

## 环境要求

- Python 3.x
- `tkinter` (通常随 Python 一起安装)
- `customtkinter` (用于现代美化界面)
- `Pillow` (用于处理图标和 `customtkinter` 图像)

## 安装与运行

### 1. 源码运行

在项目目录下执行：

```bash
python lst_generator.py
```

### 2. 使用可执行程序 (EXE)

在 `dist` 目录下可以找到 `lst_generator.exe`。
您可以直接双击运行此程序，无需安装 Python 环境。

## 操作指南

1. **选择目标目录**：点击第一个“浏览...”按钮，选择您想要列出文件的文件夹。
2. **选择保存目录**：点击第二个“浏览...”按钮，选择生成的 `.lst` 文件存放的位置。
3. **设置文件名**：在输入框中输入文件名（如 `my_list.lst`）。
4. **选项**：根据需要勾选“递归扫描子目录”。
5. **生成**：点击“立即生成 .lst 文件”按钮。

## 开发与打包

### 项目结构

- `lst_generator.py`: 核心源代码。
- `logo/`: 存放项目图标 `logo.ico`。
- `dist/`: 存放生成的 `.exe` 文件。

### 重新打包 EXE

如果您修改了代码并想重新打包，请确保已安装 `pyinstaller`，然后运行：

```bash
pyinstaller --noconfirm --onefile --windowed --icon="logo/logo.ico" --add-data "logo;logo" lst_generator.py
```
